## ✨ Features

- **3D Particle System** — Three.js golden yarn particles with mouse-reactive repulsion physics
- **Cinematic Logo Reveal** — Multi-phase GSAP entrance with scan-line sweep and glow burst
- **Custom Cursor** — Glassmorphic golden cursor with magnetic hover effects
- **Scroll Animations** — GSAP ScrollTrigger reveals, parallax, and stagger effects
- **3D Card Hover** — Vanilla-Tilt.js perspective effects on product cards
- **Interactive Tabs** — Animated textile product categories
- **Image Zoom Modal** — Click-to-zoom on all product images
- **Film Grain Overlay** — Cinematic texture overlay
- **Responsive Design** — Optimised for mobile, tablet, and desktop

## 🛠 Tech Stack

| Technology | Purpose |

| CSS3 | Dark-mode styling, glassmorphism, animations |
